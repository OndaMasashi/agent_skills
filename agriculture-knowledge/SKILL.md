# Agriculture Knowledge Skill

日本農業が直面する構造的課題と知見を体系的に整理し、農業関連の議論・分析・提案に活用するためのスキル。

## 使用場面

- 農業経営者・法人へのコンサルティング
- 農業政策の分析・提言
- スマート農業導入の検討
- 農地集約化・地域計画の策定支援
- 食料安全保障に関する議論
- 農業ビジネスモデルの検討
- 地域農業の課題整理

## 課題カテゴリ

日本農業の課題は7つのカテゴリに整理されている。各カテゴリの詳細は`references/`ディレクトリを参照。

| カテゴリ | 参照ファイル | 主な課題 |
|---------|------------|---------|
| 担い手・労働力 | [workforce.md](references/workforce.md) | 経営体数激減、高齢化、新規就農、外国人材 |
| 農地利用・基盤整備 | [farmland.md](references/farmland.md) | 農地分散、耕作放棄地、大区画化、地域計画 |
| スマート農業・FaaS | [smart-agriculture.md](references/smart-agriculture.md) | 初期投資、稼働率、季節集中、オペレーション |
| 食料安全保障・サプライチェーン | [food-security.md](references/food-security.md) | 資材高騰、価格転嫁、輸入依存、物流問題 |
| 経営管理・ビジネスモデル | [business.md](references/business.md) | どんぶり勘定、6次産業化、付加価値、資金調達 |
| 地域コミュニティ・環境 | [community.md](references/community.md) | 集落機能、獣害、環境負荷低減、中山間地域 |
| 制度・意識 | [institution.md](references/institution.md) | 補助金依存、経営継承、多様な人材、消費者理解 |

## 使用方法

1. **課題の特定**: ユーザーの質問・関心から該当するカテゴリを特定
2. **詳細参照**: 該当する`references/`ファイルを読み込み
3. **分析・提案**: 課題の構造的背景を踏まえた分析・提案を提供

## 活用例

**質問例**: 「スマート農業を導入したいが、コストが合わない」
→ [smart-agriculture.md](references/smart-agriculture.md)を参照し、初期投資問題、FaaSモデル、「春の壁・秋の壁」などの構造的課題を踏まえた提案

**質問例**: 「後継者がいない農家の農地をどうすべきか」
→ [farmland.md](references/farmland.md)と[workforce.md](references/workforce.md)を参照し、地域計画、第三者承継、農地中間管理機構などの選択肢を提示
