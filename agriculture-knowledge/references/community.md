## 集落機能の課題

### 課題42: 集落機能の低下

**現状**: 人口減少により共同作業が維持できない。

**維持困難になっている活動**:
- 水路掃除（江払い）
- 草刈り（畦畔管理）
- 農道補修
- 共同防除
- 祭り・行事

**要因**:
- 農家戸数の減少
- 高齢化による労働力低下
- 若年層の流出
- 非農家の増加

**影響**:
- 農業インフラの荒廃
- 地域の結束力低下
- 農地の管理困難

**対応策**:
- 多面的機能支払交付金の活用
- 広域化（複数集落での共同活動）
- 外部人材の活用
- 機械化・省力化

## 環境・自然の課題

### 課題43: 獣害の深刻化

**現状**: イノシシ、シカ、サル等による農作物被害が深刻。

**被害額**: 年間約150億円以上（全国）

**被害拡大の要因**:
- 耕作放棄地の増加（生息域の拡大）
- 里山の管理放棄
- 個体数の増加
- 人間への警戒心の低下

**影響**:
- 農作物の直接被害
- 営農意欲の低下
- 精神的ストレス
- 離農の一因

**対策**:
- 侵入防止柵の設置
- 捕獲（わな、銃）
- 緩衝帯の整備
- 集落ぐるみの対策

### 課題44: 環境負荷低減の要請

**現状**: 「みどりの食料システム戦略」に基づく転換が求められる。

**目標（2050年）**:
- 化学農薬使用量: 50%低減
- 化学肥料使用量: 30%低減
- 有機農業面積: 25%（100万ha）

**課題**:
- 技術的なハードル
- 収量減少リスク
- コスト増
- 労働力増

**取り組み**:
- IPM（総合的病害虫管理）
- 有機農業への転換
- 土づくり（堆肥活用）
- スマート技術による精密施肥

## 中山間地域の課題

### 課題45: 中山間地域の維持困難

**現状**: 平地に比べ条件が不利で、存続が危ぶまれる集落も。

**不利条件**:
- 傾斜地での作業効率低下
- 小区画・分散圃場
- 機械化の困難さ
- アクセスの悪さ

**課題**:
- 担い手が見つからない
- 収益性の低さ
- 集落機能の脆弱化
- 生活サービスの撤退

**支援制度**:
- 中山間地域等直接支払制度
- 多面的機能支払交付金
- 地域おこし協力隊

### 課題46: 防災・減災

**現状**: 気候変動による農業被害が激甚化。

**リスク**:
- 豪雨による浸水・土砂災害
- 台風被害
- 高温障害
- 干ばつ

**課題**:
- 農業インフラの老朽化
- ため池の安全対策
- 排水機能の不足
- 復旧の長期化

**対策**:
- 農業用ため池の防災工事
- 排水施設の整備
- 農業共済（収入保険）
- BCP（事業継続計画）

## 多面的機能の整理

農業・農村は食料生産以外にも多くの機能を持つ。

| 機能 | 内容 |
|-----|-----|
| 国土保全 | 洪水防止、土砂崩壊防止 |
| 水源涵養 | 地下水涵養、水質浄化 |
| 自然環境保全 | 生物多様性、生態系維持 |
| 良好な景観 | 農村景観、棚田 |
| 文化継承 | 伝統文化、祭り、食文化 |
| 保健休養 | グリーンツーリズム、農業体験 |
