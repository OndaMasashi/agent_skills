## 思考の基礎（OS）

### MECE（ミーシー）
**概要**: 漏れなく、ダブりなく物事を整理する考え方

- Mutually Exclusive and Collectively Exhaustive
- ロジカルシンキングの基本
- 分析や整理の際に網羅性と排他性を確保

### ロジックツリー
**概要**: 問題をツリー状に分解し、真因や解決策を論理的に探る手法

- 「売上が上がらない」→「客数減」「単価減」のように分解
- Why型（原因追求）とHow型（解決策探索）がある
- MECEを意識して分解する

### 5W1H
**概要**: When, Where, Who, What, Why, Howで情報を網羅

- 事業計画策定やヒアリング時の抜け漏れ防止に有効
- 5W2H（How much追加）、6W3H等のバリエーションあり

## 環境分析

### PEST分析
**概要**: マクロ環境（政治・経済・社会・技術）の分析

| 要素 | 分析内容 |
|-----|---------|
| Politics（政治） | 法規制、税制、政策動向 |
| Economy（経済） | 景気、金利、為替、物価 |
| Society（社会） | 人口動態、ライフスタイル、価値観 |
| Technology（技術） | 技術革新、デジタル化、特許 |

自社でコントロールできない外部環境要因を把握する。

### 5フォース分析
**概要**: 業界の収益性を決める5つの競争要因分析

1. **新規参入の脅威**: 参入障壁の高さ
2. **売り手の交渉力**: 仕入先の影響力
3. **買い手の交渉力**: 顧客の影響力
4. **代替品の脅威**: 代替製品・サービス
5. **競合間の敵対関係**: 既存競合との競争

業界の魅力度を評価し、戦略の方向性を決める。

### 3C分析
**概要**: 市場・顧客、競合、自社の分析

| 要素 | 分析内容 |
|-----|---------|
| Customer（顧客） | 市場規模、ニーズ、購買行動 |
| Competitor（競合） | 競合の戦略、強み・弱み |
| Company（自社） | 自社の経営資源、強み・弱み |

マーケティング環境分析の基本フレームワーク。

### VRIO分析
**概要**: 経営資源の競争優位性評価

| 要素 | 評価基準 |
|-----|---------|
| Value（経済価値） | 顧客に価値を提供できるか |
| Rarity（希少性） | 競合が持っていないか |
| Inimitability（模倣困難性） | 真似しにくいか |
| Organization（組織） | 活用する組織体制があるか |

4つ全てを満たすと「持続的競争優位」となる。

### バリューチェーン
**概要**: 企業の活動を「主活動」と「支援活動」の連鎖として捉える

**主活動**: 購買物流→製造→出荷物流→マーケティング・販売→サービス
**支援活動**: 全般管理、人事労務、技術開発、調達活動

付加価値がどの工程で生まれているかを分析する。

### コアコンピタンス
**概要**: 他社が真似できない核となる強み

評価基準:
1. 顧客に利益をもたらすか
2. 模倣困難か
3. 多用途に応用可能か

### 7S分析
**概要**: 組織の全体像（ハード・ソフト）分析

| 分類 | 要素 |
|-----|-----|
| ハード | Strategy（戦略）、Structure（組織）、System（制度） |
| ソフト | Shared Value（価値観）、Skill（スキル）、Staff（人材）、Style（スタイル） |

## 戦略立案

### MVV（ミッション・ビジョン・バリュー）
**概要**: 企業の使命・将来像・行動指針

| 要素 | 内容 |
|-----|-----|
| Mission（ミッション） | 企業の存在意義・使命 |
| Vision（ビジョン） | ありたい姿・将来像 |
| Value（バリュー） | 大切にする価値観・行動指針 |

### SWOT分析 / クロスSWOT
**概要**: 強み・弱み・機会・脅威の分析と戦略導出

| | 機会（O） | 脅威（T） |
|---|---------|---------|
| 強み（S） | 積極攻勢戦略 | 差別化戦略 |
| 弱み（W） | 弱点強化戦略 | 撤退・防衛戦略 |

### PPM（プロダクト・ポートフォリオ）
**概要**: 事業の資源配分を決定する枠組み

| | 市場成長率：高 | 市場成長率：低 |
|---|-------------|-------------|
| 相対シェア：高 | 花形（Star） | 金のなる木（Cash Cow） |
| 相対シェア：低 | 問題児（Question Mark） | 負け犬（Dog） |

### アンゾフの成長マトリクス
**概要**: 製品×市場の4つの成長戦略

| | 既存製品 | 新製品 |
|---|---------|-------|
| 既存市場 | 市場浸透 | 新製品開発 |
| 新市場 | 新市場開拓 | 多角化 |

### プロダクトライフサイクル
**概要**: 製品の寿命に応じた戦略転換

| 段階 | 特徴 | 戦略 |
|-----|-----|-----|
| 導入期 | 売上低、利益マイナス | 認知向上、初期顧客獲得 |
| 成長期 | 売上急増、利益増 | シェア拡大、差別化 |
| 成熟期 | 売上安定、利益最大 | 効率化、市場防衛 |
| 衰退期 | 売上減、利益減 | 撤退判断、収穫戦略 |

### イノベーター理論
**概要**: 新製品の普及プロセスと顧客層分類

1. イノベーター（2.5%）: 革新者
2. アーリーアダプター（13.5%）: 初期採用者
3. アーリーマジョリティ（34%）: 前期追随者
4. レイトマジョリティ（34%）: 後期追随者
5. ラガード（16%）: 遅滞者

**キャズム理論**: アーリーアダプターとマジョリティの間に深い溝がある。

### ビジネスモデルキャンバス
**概要**: ビジネスの全体像を1枚で可視化する9要素

1. 顧客セグメント
2. 価値提案
3. チャネル
4. 顧客との関係
5. 収益の流れ
6. リソース
7. 主要活動
8. パートナー
9. コスト構造

## マーケティング戦術

### STP分析
**概要**: ターゲット市場の選定プロセス

1. **Segmentation（細分化）**: 市場を分割
2. **Targeting（選定）**: 狙う市場を決定
3. **Positioning（位置づけ）**: 競合との差別化

### ポジショニングマップ
**概要**: 競合との差別化位置の可視化

- 独立した2軸（例：価格×品質）で配置
- 空白地帯（ブルーオーシャン）を探す
- 自社の強みを活かせるポジションを選択

### 4P（マーケティングミックス）
**概要**: 実行戦術の4要素の整合性

| 要素 | 内容 |
|-----|-----|
| Product（製品） | 製品・サービスの設計 |
| Price（価格） | 価格設定 |
| Place（流通） | チャネル・立地 |
| Promotion（販促） | 広告・PR・販売促進 |

### ペルソナ / カスタマージャーニー
**概要**: 具体的な顧客像とその体験プロセス

- **ペルソナ**: 典型的顧客の詳細プロフィール
- **カスタマージャーニー**: 顧客の思考・感情・行動を時系列で可視化
- タッチポイントごとの施策を設計

### AIDMA / AISAS
**概要**: 購買決定プロセスモデル

**AIDMA**: Attention→Interest→Desire→Memory→Action
**AISAS（Web時代）**: Attention→Interest→Search→Action→Share

### ファネル分析
**概要**: 購入に至るまでの歩留まり分析

認知→興味→検討→購入の各段階での離脱率を測定・改善。

### RFM分析
**概要**: 優良顧客の選別・分析手法

| 指標 | 内容 |
|-----|-----|
| Recency | 最新購買日（いつ買ったか） |
| Frequency | 購買頻度（何回買ったか） |
| Monetary | 購買金額（いくら買ったか） |

## マネジメント

### KGI / KPI
**概要**: 目標達成のための指標管理

- **KGI（重要目標達成指標）**: ゴール（売上、利益など）
- **KPI（重要業績評価指標）**: プロセス（商談数、成約率など）

### SMARTの法則
**概要**: 目標設定の質を高める5要素

| 要素 | 内容 |
|-----|-----|
| Specific | 具体的 |
| Measurable | 測定可能 |
| Achievable | 達成可能 |
| Related | 関連性 |
| Time-bound | 期限明確 |

### OKR
**概要**: 野心的な目標と主要な結果による管理

- **Objective**: 定性的な目標（ワクワクするゴール）
- **Key Results**: 定量的な成果指標（3〜5個）
- Google等で採用。達成率60-70%が適正。

### As is / To be
**概要**: 現状と理想のギャップ分析

- As is: 現状の姿
- To be: あるべき姿・目標
- Gap: 埋めるべき課題

### PDCAサイクル
**概要**: 継続的な業務改善サイクル

Plan（計画）→ Do（実行）→ Check（評価）→ Action（改善）

### OODAループ
**概要**: 現場判断重視の意思決定ループ

Observe（観察）→ Orient（状況判断）→ Decide（意思決定）→ Act（行動）

PDCAより高速で、変化の激しい環境に適する。

### ECRS（イクルス）の原則
**概要**: 業務改善・効率化の検討順序

1. **Eliminate（排除）**: やめられないか
2. **Combine（結合）**: まとめられないか
3. **Rearrange（交換）**: 順序を変えられないか
4. **Simplify（簡素化）**: 簡単にできないか

### バランスト・スコアカード（BSC）
**概要**: 財務以外の視点も含めた多面的な経営評価

1. 財務の視点
2. 顧客の視点
3. 業務プロセスの視点
4. 学習と成長の視点

### Will / Skill マトリクス
**概要**: 意欲と能力に応じた人材マネジメント

| | スキル高 | スキル低 |
|---|---------|---------|
| やる気高 | 委任 | 指導 |
| やる気低 | 動機づけ | 管理 |
