## バーナードの組織の3要素

### 概要
チェスター・バーナードが提唱した、公式組織が成立するための3つの要素。

### 3つの要素

| 要素 | 内容 |
|------|------|
| 共通目的 | 組織の存在意義・目的をメンバー全員が共有している |
| 貢献意欲（協働意欲） | 成員が進んで時間・労力・知識を組織のために使おうとする気持ち |
| コミュニケーション | 目的・方針・指示・フィードバックを相互に伝達する仕組み |

### ポイント

- 公式組織は「意識的に調整された2人以上の人々の活動と諸力のシステム」
- 3要素のいずれか一つ欠けると組織不全になる
- コミュニケーションは他の2要素を実現する手段として特に重要

### 中小企業での具体例

- **共通目的**：ミッションを全員が理解
- **貢献意欲**：成長機会・公正な評価・裁量のある働き方
- **コミュニケーション**：週次MTG、1on1、チャット・タスク管理ツール

## 採用戦略と定着戦略

### 全体設計の3フェーズ

| フェーズ | 内容 |
|----------|------|
| 採用前・選考 | 採用要件明文化、ネガティブ情報も開示、採用ブランディング |
| 入社直後 | オンボーディング計画、メンター配置、1on1面談 |
| 定着・成長 | 定着率管理、キャリア面談、従業員サーベイと環境改善 |

### ミスマッチ防止（採用前・選考）

- **構造化面接**：質問項目・評価基準を事前定義、面接官トレーニング
- **コンピテンシー評価・適性検査**：行動特性・価値観のフィット確認
- **カジュアル面談・社員座談会**：現場社員との対話機会
- **体験入社・インターンシップ**：実際の仕事・雰囲気を経験
- **リファラル採用**：社員紹介でカルチャーフィットしやすい人材へアクセス
- **リファレンスチェック**：前職での行動特性・評価を第三者から確認

**重要**: 企業が候補者を見るだけでなく、候補者にも十分に自社を見てもらう設計

### オンボーディング（入社直後）

- 入社初日〜3か月のオンボーディング計画を明文化
- ビジョン・ミッションをワークショップ形式で本人のキャリアと結びつけ
- 上司・メンター側への「受け入れ教育」
- 週〜月1回の1on1面談で期待値すり合わせ・不安の吸い上げ

### 定着戦略（中長期）

- 定着率・離職率を年代別・部門別・入社経路別に定量管理
- 若手早期離職防止策：メンター制度、キャリア面談、ジョブローテーション
- 部門横断コミュニケーション：社内イベント、シャッフルランチ
- キャリアパス・期待役割の定期対話
- 従業員サーベイや1on1で出た不満・アイデアから環境改善

### 中小企業で取り組みやすい具体施策

1. 採用要件シートと面接評価シートを全面接官で共通利用
2. カジュアル面談＋現場社員座談会を標準フローに組み込み
3. 入社後3か月のオンボーディングチェックリスト作成
4. 新入社員1人につき1人のメンターを指定、月1回面談を半期継続
5. 年1〜2回の従業員アンケートと改善アクション公開

## インターナルマーケティング

### 概要
従業員を「社内の顧客（内部顧客）」と捉え、満足度やエンゲージメントを高めることで、顧客満足と業績向上につなげる取り組み。

### 目的と期待効果

| 目的 | 効果 |
|------|------|
| 従業員満足度向上 | 離職率低下、エンゲージメント向上 |
| 顧客満足度向上 | モチベーション高い従業員による良い接客・対応 |
| 業績・生産性向上 | サービス・プロフィット・チェーンによる売上・利益改善 |

### 主な施策

| 分類 | 施策例 |
|------|--------|
| 教育・研修 | ビジョン共有研修、商品知識研修、接客スキル研修 |
| 内部コミュニケーション | 社内報、社内SNS、タウンホールミーティング |
| 報酬・評価制度 | 顧客志向の行動評価、納得感のある処遇設計 |
| エンゲージメント施策 | 表彰制度、社内イベント、ウェルビーイング施策 |
| 働きやすい環境整備 | 勤務制度見直し、福利厚生充実、柔軟な働き方 |

### 類似概念との違い

- **エクスターナルマーケティング**：顧客（市場）向けの対外的マーケティング
- **インターナルブランディング**：企業理念・ブランド価値の従業員への浸透
- **インターナルマーケティング**：従業員満足と行動変容を促す仕組みづくり全般

## 外国人採用

### 主な就労系ビザの種類

| 在留資格 | 対象業務 | 特徴 |
|----------|---------|------|
| 技術・人文知識・国際業務 | ITエンジニア、事務、企画、通訳等ホワイトカラー系 | 大卒or実務経験3年以上、学歴・キャリアとの整合が必要 |
| 特定技能（1号・2号） | 建設、製造、外食、宿泊、介護、農業等の現場作業 | 1号は最長5年・家族帯同不可、2号は更新無制限・家族帯同可 |
| 技能 | 外国料理人、調理師、職人等 | 特定の熟練技能が必要 |
| 高度専門職 | ポイント制を満たす高度人材 | 在留期間優遇、家族帯同しやすい |
| 経営・管理 | 会社経営者・役員 | 投資額・事業計画等の条件あり |

### 企業側の必須チェックポイント

1. **在留資格と仕事内容の適合**
   - 在留資格の範囲を超えた仕事は不法就労・不法就労助長
   - 例：技術・人文知識・国際業務で採用→単純作業のみはNG

2. **在留カードの確認・管理**
   - 在留資格・在留期間・就労制限の有無を確認、コピー保管
   - 在留期限の管理台帳を作り定期チェック（月1回等）

3. **労働条件は日本人と同等以上**
   - 最低賃金法・労基法は国籍に関係なく適用
   - 「日本人と同等以上の報酬か」がビザ審査でも見られる

4. **法定手続き**
   - 雇入れ・離職時にハローワークへ届出義務
   - 社会保険の加入条件を満たせば日本人と同様に手続き

### ありがちな落とし穴

- 「就労ビザがあるからOK」と仕事内容を変えてしまう
- 留学生アルバイトの資格外活動（週28時間）を超過
- コミュニケーション・文化面のケア不足（多言語での説明不足）

### 中小企業の実務ステップ

1. 人手不足分野と適合する在留資格を整理
2. 学歴・職歴と仕事内容が要件に合うか行政書士等と確認
3. 在留資格認定証明書の申請、許可後に雇入れ
4. 在留カード台帳・期限管理、届出、多言語化等の運用体制整備

## 評価制度とMBO

### 評価制度と組織開発の関係

評価制度は「どんな行動・成果を良しとする組織か」を示すレバー。ビジョン・戦略・期待行動とつながるほど、エンゲージメントや一体感が高まる。

### 公平性・透明性・納得性

| 原則 | 内容 |
|------|------|
| 透明性 | 評価基準・プロセス・評価者を誰にでもわかる形でオープンに |
| 公平性 | 同じ役割・成果には同じ評価、特定個人を有利・不利に扱わない |
| 納得性 | 「そう評価される理由は筋が通っている」と本人が感じる状態 |

### 具体策

- 評価基準の明文化・公開
- 評価者トレーニング
- 複数評価者やレビュー会議（キャリブレーション）
- 評価結果のフィードバック面談

### MBO（目標管理制度）

**概要**: 期初に上司と部下が目標と評価基準を合意し、期末に達成度で評価する仕組み

**メリット**
- 数値・明文化目標に基づく客観性の高い評価
- 目標・基準を共有するため透明性が高く納得感を得やすい
- 個人目標を組織目標と連動させやすい

**デメリット・注意点**
- 数値化しにくい職務は評価が主観に寄りやすい
- 短期数値目標偏重で長期能力開発・協働が軽視されるリスク
- 「形だけの目標シート」になりやすい

### MBO設計のコツ

| 観点 | 内容 |
|------|------|
| 目標と評価基準の具体化 | SMART原則、定量・定性のバランス、達成度と評価ランクの対応表 |
| 評価プロセスの透明化 | 期初・中間・期末の面談を制度化、理由や観察事実をフィードバック |
| 公平性の担保 | 職種・グレードごとに項目・ウエイト定義、評価者研修・評価会議 |
| 成果と成長の両立 | 成果目標（What）＋行動・能力目標（How）、学習・挑戦テーマ枠 |

### 組織開発とつなげる運用

- **期初**：経営方針・事業計画を説明し、部門・個人目標をすり合わせ
- **中間**：進捗だけでなく組織課題・改善アイデアも対話、組織学習の素材収集
- **期末**：評価フィードバックに次期成長テーマ・キャリア期待の対話を組み込み
