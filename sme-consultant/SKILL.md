# SME Consultant Skill

中小企業診断士試験で問われる経営知識・フレームワークを体系的に整理し、経営課題の分析・解決に活用するためのスキル。

## 使用場面

- 経営戦略の策定・評価
- マーケティング施策の立案
- 組織・人事課題の分析
- 生産管理・業務改善
- 財務分析・投資意思決定
- 事業承継・M&Aの検討
- ビジネスプラン作成支援

## 知識カテゴリ

中小企業診断士の知識体系は以下のカテゴリに整理されている。

| カテゴリ | 参照ファイル | 主な内容 |
|---------|------------|---------|
| 思考フレームワーク | [frameworks.md](references/frameworks.md) | MECE、ロジックツリー、環境分析、戦略立案、マーケティング、マネジメント |
| 事例I：組織・人事 | [case1-organization.md](references/case1-organization.md) | 経営戦略、組織構造、事業承継、人的資源管理、組織論 |
| 事例II：マーケティング | [case2-marketing.md](references/case2-marketing.md) | サービス業、ブランド戦略、顧客関係、流通・チャネル、価格戦略 |
| 事例III：生産・運営 | [case3-operations.md](references/case3-operations.md) | 生産管理、レイアウト、在庫管理、現場改善、設備保全 |
| 事例IV：財務・会計 | [case4-finance.md](references/case4-finance.md) | 財務分析、CVP分析、管理会計、投資決定、企業価値評価 |

## 使用方法

1. **課題の特定**: ユーザーの質問・関心から該当するカテゴリを特定
2. **詳細参照**: 該当する`references/`ファイルを読み込み
3. **フレームワーク適用**: 適切なフレームワーク・指標を用いて分析・提案

## 活用例

**質問例**: 「新規事業を検討しているが、どう分析すべきか」
→ [frameworks.md](references/frameworks.md)のSWOT分析、アンゾフの成長マトリクスを参照

**質問例**: 「組織が縦割りで部門間連携ができない」
→ [case1-organization.md](references/case1-organization.md)の組織構造、バウンダリスパナーを参照

**質問例**: 「投資判断の方法を教えてほしい」
→ [case4-finance.md](references/case4-finance.md)のNPV、IRR、WACCを参照
