# DOCX Word Count

このスキルは、Markdown (.md) ファイルから、Microsoft Word でカウントされる文字数に近い数値を算出します。
見出し、リスト、リンク、コードブロックなどの MD 特有の記法を解析から除外するため、提出書類の文字数確認などに役立ちます。

## カウントの仕様

1. **以下の要素はカウントから除外されます**:
   - MD 記号（`#`, `*`, `-`, `>`, `1.` など）
   - コードブロック (` ``` `) とその中身
   - インラインコード (`` ` ``)
   - 画像記法 (`![alt](url)`)
   - リンクの URL 部分（表示テキストのみカウント）
   - HTML タグ

2. **カウント結果**:
   - **文字数 (スペースなし)**: すべての空白、改行を除去した文字数
   - **文字数 (スペースあり)**: 改行のみを除去し、スペースは含めた文字数

## 使用方法

スクリプト `scripts/count.py` を使用してカウントを実行します。

```bash
python scripts/count.py path/to/your/file.md
```

## 使用例

```bash
# 実行例
$ python scripts/count.py draft.md
ファイル: draft.md
文字数 (スペースなし): 1250
文字数 (スペースあり): 1275
```
